4 // The BoundedList provides a doubly-linked list similar to std::list, however,
5 // it is limited in the amount of values that can be pushed in the list. Moreover, nodes can
6 // be moved from one std::list to another in constant time, but in BoundedList
7 // this is done in linear time, since nodes are copied from one BoundedList to another.
8 // Node storage is specifically coupled to a BoundedList.
9 // The usecase for using BoundedList is that values can be inserted in the middle of
10 // the list in constant time.
