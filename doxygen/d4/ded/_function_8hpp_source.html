4 // Function is an object which can contain any callable function object, e.g. free function pointers
5 // and lambda expressions. A Function is declared by providing the function signature and optionally
6 // extra storage space for the function. For example, Function<void()> f; declares a function object
7 // f that has void return and no parameters. Function<int(char x)> g; declares a function object g
8 // that has an int return and one parameter of type char.
9 // Examples:
10 //
11 // Function<void()> f = [this]() { Trigger(); };
12 // if (f)
13 // f(); // call the lambda function which calls this->Trigger
14 // f = nullptr; // Clear the function
15 //
16 // Since no heap is used, all data used by the function object must be stored inside Function itself.
17 // The amount of data necessary is not known upfront, so the Function is parameterized with an
18 // ExtraSize parameter to reserve storage. For example, this lambda expression requires no storage:
19 // []() {} while this expression requires 8 bytes of extra storage: [this, &x]() { DoSomething(x); }.
20 //
21 // Function<void()> f = [this, &x]() { DoSomething(x); }; // Compile error, too much storage is needed
22 // Function<void(), 8> g = [this, &x]() { DoSomething(x); }; // Ok.
