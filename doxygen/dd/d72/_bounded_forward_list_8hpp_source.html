4 // The BoundedForwardList provides a singly-linked list similar to std::forward_list, however,
5 // it is limited in the amount of values that can be pushed in the list. Moreover, nodes can
6 // be moved from one std::forward_list to another in constant time, but in BoundedForwardList
7 // this is done in linear time, since nodes are copied from one BoundedForwardList to another.
8 // Node storage is specifically coupled to a BoundedForwardList.
9 // The usecase for using BoundedForwardList is that values can be inserted in the middle of
10 // the list in constant time.
